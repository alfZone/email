# How to install

For complete use of email sender, follow the steps below.

## Gmail Configuration Steps

1. **Enable 2-Step Verification** on your Google account

2. **Generate an App Password**:
   - Access: https://myaccount.google.com/apppasswords
   - Select "Other" as the application type
   - Give it a name (e.g., "My Website")
   - Copy the generated 16-digit password

3. **Install PHPMailer via Composer**:
   ```bash
   composer require phpmailer/phpmailer
